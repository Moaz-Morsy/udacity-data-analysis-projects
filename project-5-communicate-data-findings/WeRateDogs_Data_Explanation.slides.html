Dataset Overview¶

This data set contains information about 1300 dogs collected from the tweet archive of Twitter user @dog_rates, also known as WeRateDogs and The tweet image predictions which is hosted on Udacity's servers, this data contains:¶

  • tweet_id: is the last part of the tweet URL after "status/" → https://twitter.com/dog_rates/status/889531135344209921
  • timestamp: time when tweet is posted
  • time_of_day: which part of day when the tweet is posted
  • text: the tweet itself
  • jpg_url: url of image attached to the tweet
  • name: name of the dog
  • stage: which breed does the dog belong to, ex:doggo, pup, woof
  • dog_breed: which breed does the dog belong to
  • img_num: the image number that corresponded to the most confident prediction (numbered 1 to 4 since tweets can have up to four images)
  • algorithm_conf: how confident the algorithm is in its prediction
  • prediction algorithm: the chosen algorithm
  • full_rating: rate of the dog chosen by dog's owner
  • rating_numerator: numerator part of the rating
  • retweet_count
  • favorite_count
